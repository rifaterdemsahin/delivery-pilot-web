# Environment Documentation

This directory contains documentation for the various environments used in the Delivery Pilot Web project.

## Available Environments

- **[Local Development](local.md)**: How to set up and run the project on your own machine.
- **[GitHub Codespaces](codespaces.md)**: How to use the cloud-based development environment.
- **[GitHub](github.md)**: General GitHub workflow, issues, and PRs.
- **[GitHub Pages](github_pages.md)**: How the production site is hosted and deployed.

## Quick Start

For most contributors, we recommend using **Local Development** for offline work or **Codespaces** for quick edits.
