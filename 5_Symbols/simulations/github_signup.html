Process Checklist ‚úÖ

Complete each step in order. Check the box when done.

1. Navigate to GitHub
Open a new tab and go to github.com.
2. Click 'Sign up'
Find the 'Sign up' button in the top right corner.
3. Enter Email
Enter your valid email address when prompted by the terminal-like UI.
4. Create Password
Create a strong password (keep it safe!).
5. Choose Username
Select a unique username. This will be your developer identity.
6. Verify Account
Solve the visual puzzle to prove you are human.
7. Email Verification Code
Check your email inbox for the launch code and enter it on GitHub.
8. Navigate to Target Repo
Go to the target repository URL provided by your instructor (or use this demo repo).
9. Copy Clone URL
Click the green 'Code' button and copy the HTTPS URL.
10. Clone in Terminal
Open your terminal and run: `git clone [pasted-url]`
